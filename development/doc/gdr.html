Overview

The GNU dr reminds you about birthdays and celebrations according to the data of the input files. By default, GNU dr works with the system's date.

GNU dr is protected by the GNU General Public License.

GNU dr was written by Efstathios Chatzikyriakidis.

Invoking

The format for running the gdr program is:

`gdr [OPTIONS] -f FILE -c FILE'

The gdr program supports the following options:

-d DATE
Work with the DATE (dd/mm).


-f FILE
The friends list FILE.


-c FILE
The celebrations list FILE.


-e
Explain what is being done.


-h
Display a short help and exit.


-v
Print version information and exit.


-a
Print author information and exit.


-l
Print the short version of GNU/GPL and exit.

Environment Variables

The behavior of gdr is affected by the following environment variables.

GDR_DATE
Specifies the date (dd/mm).
